# Privacy Policy

**RouteFlow Map** is a Power BI custom visual for rendering origin-destination flow maps.

## Data Collection

This visual **does not** collect, store, or transmit any personal or user data.

## Third-Party Services

Map tiles are loaded from [Mapbox](https://www.mapbox.com/) using an access token provided by the user. Mapbox's own privacy policy applies to map tile requests: https://www.mapbox.com/legal/privacy

## Contact

For questions, open an issue on this repository.
