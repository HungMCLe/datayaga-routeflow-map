# Support

## Reporting Issues

For bugs or feature requests, please [open a GitHub issue](../../issues) on this repository.

## Requirements

- Power BI Desktop or Power BI Service
- A valid [Mapbox access token](https://account.mapbox.com/access-tokens/)
- Data with origin/destination coordinates and route strings

## Quick Start

1. Import the `.pbiviz` file into Power BI
2. Add your Mapbox token in the visual's format settings
3. Drag route string, leg origin/destination codes into categories
4. Drag lat/lon measures and value measures into values
